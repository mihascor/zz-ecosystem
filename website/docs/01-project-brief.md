# Project Brief

## Название

«Запретные знания»

## Тип проекта

Контентный сайт-блог для публикации статей, изображений и структурированного чтения материалов.

## Цель продукта

Создать локально работающий сайт-блог, который позволяет размещать авторские материалы заказчика в формате статей и удобно читать их последовательно или выборочно.

## Основные пользователи

- Читатели, которым нужен удобный доступ к статьям по категориям, подкатегориям и тегам.
- Автор или владелец проекта, который публикует материалы через локальные Markdown/MDX-файлы.
- Разработчик проекта, который поддерживает сайт как самостоятельную часть экосистемы `zz-ecosystem`.

## Назначение сайта

Сайт должен предоставлять:

- список опубликованных статей;
- отдельные страницы статей;
- навигацию по категориям, подкатегориям и тегам;
- левый сайдбар с деревом статей;
- последовательное чтение через переходы к предыдущей и следующей статье;
- переключение светлой, тёмной и системной темы;
- локальный запуск на компьютере разработчика.

## Место в экосистеме

Проект `website` является самостоятельным проектом внутри общей папки экосистемы:

```text
zz-ecosystem/
├─ website/
├─ rag-system/
├─ ai-agent-chat/
├─ video-generator/
├─ social-publisher/
└─ docs/
```

Сайт не должен зависеть от будущих сервисов. Возможная интеграция с другими проектами экосистемы может появиться позже через API, ссылки, поддомены или общую авторизацию.

## Границы текущего этапа

В текущий этап входит локальная версия сайта-блога на Astro, TypeScript, Markdown/MDX и Tailwind CSS.

В текущий этап не входит:

- подключение домена;
- деплой;
- авторизация;
- личный кабинет;
- платежи;
- RAG-система;
- ИИ-агент;
- векторная база данных;
- генерация видео;
- публикация в соцсети;
- админ-панель через браузер;
- подключение внешней LLM.
