# Development Plan

## Этап 1. Документация проекта

- Разложить техническое задание по проектным документам.
- Зафиксировать требования, архитектуру, модель данных, внутренние контракты и решения.

## Этап 2. Инициализация Astro-проекта

- Создать базовый Astro-проект.
- Подключить TypeScript.
- Настроить npm-скрипты для разработки, сборки и предпросмотра.
- Добавить Tailwind CSS.
- Добавить базовые файлы `README.md`, `robots.txt`, favicon.

## Этап 3. Контентная модель

- Настроить `src/content.config.ts`.
- Создать коллекцию `articles`.
- Описать обязательные поля статьи.
- Добавить несколько тестовых статей.
- Подготовить папку `public/images/articles/`.

## Этап 4. Утилиты данных

- Реализовать получение опубликованных статей.
- Реализовать сортировку по `order` и дате.
- Реализовать группировку по категориям и подкатегориям.
- Реализовать индекс тегов.
- Реализовать предыдущую и следующую статью.

## Этап 5. Базовые страницы

- Создать главную страницу `/`.
- Создать список статей `/articles`.
- Создать страницу статьи `/articles/[slug]`.
- Создать страницы категорий `/categories/[category]`.
- Создать страницы подкатегорий `/subcategories/[subcategory]`.
- Создать страницы тегов `/tags/[tag]`.

## Этап 6. Компоненты интерфейса

- Реализовать `Header`.
- Реализовать `Sidebar`.
- Реализовать `SidebarToggle`.
- Реализовать `ThemeToggle`.
- Реализовать `ArticleCard`.
- Реализовать `ArticleNavigation`.
- Реализовать `Breadcrumbs`.
- Реализовать `TagList`.
- Реализовать `CategoryTree`.

## Этап 7. Темы и состояние интерфейса

- Настроить CSS-переменные для светлой и тёмной темы.
- Реализовать переключение светлой, тёмной и системной темы.
- Сохранять выбранную тему в `localStorage`.
- Реализовать скрытие и повторное открытие сайдбара.
- Сохранять состояние сайдбара в `localStorage`.
- Добавить разворачивание и скрытие всех пунктов дерева навигации.

## Этап 8. Адаптивность и чтение

- Настроить читаемые размеры и отступы для длинных статей.
- Проверить отображение на десктопе, планшете и мобильном устройстве.
- Настроить мобильное поведение сайдбара.

## Этап 9. SEO и метаданные

- Добавить уникальные `title`.
- Добавить `description` для страниц статей.
- Добавить Open Graph-разметку.
- Проверить человекочитаемые URL.
- Подготовить возможность генерации карты сайта при необходимости.

## Этап 10. Проверка и приёмка

- Запустить `npm run dev`.
- Проверить открытие главной страницы.
- Проверить страницы статей, категорий, подкатегорий и тегов.
- Проверить скрытие неопубликованных статей.
- Проверить навигацию предыдущая/следующая статья.
- Проверить сохранение темы и состояния сайдбара после перезагрузки.
- Запустить `npm run build`.
- Обновить README с инструкциями запуска и добавления контента.
